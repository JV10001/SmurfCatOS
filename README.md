# SmurfCatOS
My operating system based off the viral meme smurfcat! It includes some features including draggable windows, open and close buttons, a clicker app, a note taking app, and a top bar with a clock. If you click on the open button/app icon it can do a few cool things like bringing the windiw to the middle to prevent it going out of the website and being unable to be dragged, it can open the window if it is closed, if it's the app icon it will be highlighted with a blue border unless you close the window, it can bring the window it corresponds to the front this goes for clicking on the window to if you are able. If you reload your tab there is a save fearure on this, so all your notes and clicks done on the clicker app will be saved. On the notebook app when you type on it it will show you when you last edited your notes!!! The backround of the OS is based off of SmurfCats backround in the meme though it is probably hard to see. The red smurfcat in the top right of the window allows you to close it, and the smurfcat that has the king clothes in the top left is the header where you can drag it. If you click on the video the welcome screen presents you with and urges you to tap you should click it for a funny cat video!!! That is all the features that I have added, hopefully I will add more soon!!!
